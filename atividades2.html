explicando o CSS


Uma regra CSS é uma instrução que define como um ou mais elementos HTML devem ser estilizados. Ela é composta por duas partes principais: o seletor e o bloco de declaração, que por sua vez contém propriedades e seus respectivos valores. Anatomia de uma Regra CSS

Uma regra CSS é formada por:

Seletor: que indica qual elemento HTML será estilizado

Propriedade: define o que será modificado (cor, tamanho, fonte, etc.)

Valor: especifica como a propriedade será aplicada

Exemplo de uma regra CSS:

p { color: red; font-size: 18px; }

Explicação:

Seletor: p

Propriedades e valores:

  • color: red; deixa o texto vermelho
  • font-size: 18px; aumenta o tamanho da fonte

    • passando o moude em uma caixa e ela muda e cor
